# Adding a New Writing Post

Use `understanding-motor-control.html` as the current static post template.

1. Create a new HTML file in `/blog`.
2. Copy the template and update the title, description, keywords, date, tags, and article body.
3. Add a matching card to `blog.html` so the post appears on the Writing page.
4. Check the page on desktop and mobile widths.

Keep the global identity consistent:

- Use "researcher, educator, and applied scientist" when a short professional description is needed.
- Use "Writing" rather than "Blog" in visible navigation.
- Frame programming and analysis as research methods.
- Avoid old resume-oriented or role-hunting language.

Post tags should stay specific and restrained, for example:

- Motor control
- Proprioception
- Research methods
- Analysis
- Teaching
